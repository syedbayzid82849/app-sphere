# APP-SPHERE — App Store

## Description

APP-SPHERE is a modern app discovery platform where users can explore, search, and install their favorite applications. Browse trending apps, view detailed app information with ratings and reviews, and manage your installed apps — all in one place.

## Features

- Browse all apps with live search
- View detailed app info with rating chart
- Install & uninstall apps (saved to localStorage)
- Sort installed apps by download count
- Fully responsive for all devices

## Technologies

- **React** — UI library
- **React Router** — client-side routing
- **Tailwind CSS** — styling
- **Recharts** — rating bar chart
- **React Hot Toast** — notifications
- **Vite** — build tool
